Sintaxis Básica y variables

Declaración de variables

A la hora de declarar una variable existen diferentes formas de hacerlo


Var

Declaración de variable con alcance global o local

Let

Declaración devariable con alcance local y bloqueado

Const

Declaración de variable constante con alcance local y bloqueado

Tipos de datos

Tipos de datos Significado Definición Ejemplo
String Secuencia de caracteres que se utilizan para representar un texto Se pueden definir usando ' o " "Hola mundo"
Number Un valor numérico usado paraa representar números enteros o decimales Se pueden definir utilizando números enteros o decimales 5, 2.5, -5
Boolean Es un valor lógico que puede ser o verdadero o falso Se utilizan para representar condiciones lógicas True, False
Array Es una colección de valores que se pueden acceder mediante un índice Se definen usando corchetes y separando elemntos con comas [ 1, 2, 3]
Object Es una colección de pares clave-valor que se pueden acceder mediante clave Se definen usando llaves y separando los pares clave-valor con con comas { nombres "Juan", edad "30" }

Estructuras de control

Condicionales

Se utilizan para ejecutar un bloque de código si se cumple una condición.

  • if: Condición simple.
  • else if: Condición múltiple.
  • else: Condición alternativa.
  • switch: Selección múltiple.

Bucles

Se utilizan para ejecutar un bloque de código repetidamente.

  • for: Bucle con contador.
  • while: Bucle con condición.
  • do while: Bucle con condición al final.

Funciones

Se utilizan para agrupar un bloque de código que se puede ejecutar varias veces.

  • function: Declaración de función.

Parámetros

Valores que se pasan a la función

Return

Valor que se devuelve desde la función.

Manipulación del DOM

Seleccionar elementos

Existen métodos para seleccionar elementos en un documento de HTML

  • document.querySelector(): Selecciona un elemento según un selector CSS.
  • document.getElementById(): Selecciona un elemento según su ID.

Modificar contenido

También se utlizan propiedades y métodos para modificar el contenido de los elementos

  • innerHTML: Contenido HTML del elemento.
  • textContent: Contenido de texto del elemento.

Cambiar estilos

A su vez, también existen propiedades y métodos para cambiar los estilos de los elementos

  • element.style: Propiedades de estilo del elemento.

Agregar o eliminar clases

hay métodos que son usados para agregar o eliminar clases CSS de los elementos

  • classList.add(): Agrega una clase CSS al elemento.
  • classList.remove(): Elimina una clase CSS del elemento.

Eventos JavaScript

Escuchar eventos

  • onclick: Evento de clic en un elemento.
  • onmouseover: Evento de paso del mouse sobre un elemento.
  • onkeydown: Evento de pulsación de una tecla.
  • addEventListener(): Agrega un evento a un elemento.

Prevenir acciones

  • event.preventDefault(): Previene la acción predeterminada del evento.
